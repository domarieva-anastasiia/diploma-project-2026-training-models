# diploma-project-2026-training-models
Репозиторій навчання моделей SRGAN для дипломної роботи. Тут розміщено ноутбуки Google Colab, які реалізують навчальний процес із трьома різними архітектурами дискримінаторів (BASE, GAP і PatchGAN). Також розміщено журнали навчання із показниками втрат та метриками.

Головна мета проєкту — дослідити та порівняти вплив різних конфігурацій дискримінатора на якість відновлення високочастотних деталей, текстур та загальну стабільність навчання моделі SRGAN. 

Усі моделі використовують спільну архітектуру генератора, але тренуються з трьома різними дискримінаторами:
1. BASE Discriminator — класична архітектура дискримінатора SRGAN, побудована на послідовних згорткових шарах (Convolutional), нормалізації батчів (Batch Normalization) та активації LeakyReLU з фінальними повнозв'язними шарами.
2. GAP Discriminator — модифікація, де замість щільних повнозв'язних шарів наприкінці мережі використовується шар глобального усереднення пулінгу (Global Average Pooling). Це дозволяє значно зменшити кількість параметрів моделі та знизити ризик перенавчання.
3. PatchGAN Discriminator — марковський дискримінатор, який оцінює реалістичність зображення не цілком, а на рівні локальних ділянок (патчів). Орієнтований на краще уловлювання чітких текстур та дрібних деталей.
