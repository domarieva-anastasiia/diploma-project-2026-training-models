# diploma-project-2026-training-models
Цей репозиторій містить аналітичні матеріали, графіки, логи тренування та візуальні результати порівняльного аналізу трьох модифікацій архітектури SRGAN.

Мета цього репозиторію — продемонструвати динаміку навчання моделей та наочно порівняти якість відновлення текстур з використанням трьох різних дискримінаторів: **BASE**, **GAP** та **PatchGAN**.

Усі моделі використовують спільну архітектуру генератора SRResNet, але тренуються з трьома різними дискримінаторами:
1. BASE Discriminator — класична архітектура дискримінатора SRGAN, побудована на послідовних згорткових шарах, нормалізації батчів Batch Normalization та активації LeakyReLU з фінальними повнозв'язними шарами.
2. GAP Discriminator — модифікація, де замість щільних повнозв'язних шарів наприкінці мережі використовується шар глобального усереднення пулінгу (Global Average Pooling). Це дозволяє значно зменшити кількість параметрів моделі та знизити ризик перенавчання.
3. PatchGAN Discriminator — марковський дискримінатор, який оцінює реалістичність зображення не цілком, а на рівні локальних ділянок (патчів). Орієнтований на краще розпізнавання чітких текстур та дрібних деталей.
