# diploma-project-2026-training-models
Репозиторій навчання моделей SRGAN для дипломної роботи. Тут розміщено ноутбуки Google Colab, які реалізують навчальний процес із трьома різними архітектурами дискримінаторів (BASE, GAP і PatchGAN). Також розміщено журнали навчання із показниками втрат та метриками.
